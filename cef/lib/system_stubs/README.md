# PSP Stubs

This is a PSPSDK export file with all the PSP system stubs that are available on the Adrenaline environment.

This was possible due to the incredible work of the PSP community so far.

The export file was generated based on the XML files of the amazing [PSPLibDoc](https://github.com/pspdev/psplibdoc) project.

## Building

To build the hole thing, is just one make invocation:

```sh
make
```

But, it is quite massive process, so it is recommended to build only the targets you want/need, as the example below:

```sh
make .pspsemaphore .pspkdebug_kernel
```
